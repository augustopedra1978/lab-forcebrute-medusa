# Laboratório: Força-Bruta / Password-Spray com Kali + Medusa (Ambiente Isolado)

> **Aviso importante:** Todo o conteúdo deste repositório foi preparado **exclusivamente** para uso em **laboratórios isolados** (ex.: VMs Host-Only / Internal Network). **Não execute** qualquer comando contra sistemas de terceiros ou em redes de produção.

## Objetivo
Simular ataques de força-bruta e password-spraying em ambiente controlado (Kali → Metasploitable2 / DVWA) para entender técnicas, coletar evidências e propor contramedidas.

## Entregáveis
- Relatório técnico (modelo em `/docs/template_relatorio.md`)
- Wordlists usadas (pequenas, em `/wordlists/`)
- Scripts utilitários (em `/scripts/`) — geração de wordlists e parser de logs
- Capturas de tela (opcional) em `/images/`
- Exemplo de resultados em `example_results.csv`
- README.md (este arquivo) explicando ambiente, metodologia e recomendações
